你是一位資深 HTML5 遊戲工程師。
請使用以下技術製作一款完整的橫向卷軸 RPG 打寶遊戲。

技術要求：

- HTML5
- CSS3
- JavaScript ES6
- Canvas API
- 不使用任何外部框架
- 所有程式碼可以直接開啟 index.html 執行

遊戲需求：

1. 玩家左右移動
2. 跳躍
3. 重力
4. 慣性
5. 地板碰撞
6. 平台碰撞
7. 近戰攻擊(即時戰鬥，非回合制)
8. 至少兩種怪物(例如史萊姆、骷髏)
9. 怪物 AI 巡邏
10. 怪物受到攻擊、死亡
11. 戰利品掉落系統(打寶核心機制)
12. 戰利品稀有度分級(普通 / 稀有 / 史詩 / 傳說)
13. 背包系統
14. 裝備系統(武器 / 防具，穿上後提升屬性)
15. 角色屬性(HP、攻擊力、防禦力)
16. 經驗值與升級系統
17. 金幣系統
18. 玩家受傷
19. 玩家死亡與重生
20. 關卡地圖(地板、浮空平台、深淵)
21. Camera Follow(攝影機跟隨玩家)
22. 世界邊界
23. 音效
24. 背景音樂
25. 暫停功能

架構要求：
project/
index.html
style.css
game.js
player.js
enemy.js
item.js
physics.js
collision.js
level.js
ui.js
遊戲素材/

** 遊戲素材使用 ./遊戲素材提供的素材包製作 **

所有功能請拆成模組。
不要全部寫在 game.js。
程式碼請加入完整註解。
所有 class 命名符合 ES6。
最後提供完整專案。
